# samsungistop

Безвредный пранк для фанатов iPhone.

## Фишки

- Спам смешными ошибками
- Поиски смешных вещей в Google
- Глюки на экране
- Зеленый код на экране
- Триггер при закрытии окна/попытке убить через **Диспетчер задач**
